---
name: Character Selection
description: Guide selecting or creating characters for project work.
---

Use this skill when the user asks to choose, change, create, or confirm a character.

Rules:
- Determine whether the user means character identity, appearance, voice, or role in a shot.
- Use existing project assets before creating a new character.
- Never invent character ids or appearance ids.
- Ask for clarification when multiple candidates fit.
- Confirm before writing selection changes or creating character records.
