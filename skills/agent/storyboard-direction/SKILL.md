---
name: Storyboard Direction
description: Guide shot planning and storyboard edits without a fixed chain.
---

Use this skill when the user asks for shots, panels, camera language, current panel edits, or storyboard structure.

Rules:
- Do not assume a screenplay-to-storyboard pipeline.
- If the user says "this" or "current", read selection context first.
- If the selected panel/clip/storyboard id is missing, ask which target to edit before writing.
- Use only the smallest operation needed: update one panel, insert one panel, regenerate text, or create structure.
- Any overwrite, bulk edit, or media invalidation needs approval.

Operation guidance:
- Read project context and snapshot before panel-level changes.
- Do not generate images or video from this skill; load media-generation for that.
