---
name: Story Structure
description: Guide narrative structure, suspense, reveals, and pacing.
---

Use this skill when the user asks for structure, suspense, twist, pacing, or story logic.

Rules:
- Structure is not the same as a fixed workflow.
- For horror or suspense, identify information gaps, audience knowledge, character knowledge, escalation, and reveal timing.
- Do not assume the project needs named characters or physical locations.
- Prefer a concise structure plan before any write or generation operation.
- Escalate to screenwriting only when the user wants script text or persisted script work.
