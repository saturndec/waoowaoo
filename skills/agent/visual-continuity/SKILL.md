---
name: Visual Continuity
description: Guide consistent visual choices across assets and panels.
---

Use this skill when the user asks for consistent characters, locations, style, or visual continuity.

Rules:
- Inspect existing assets before suggesting replacements.
- Do not invent asset ids or claim a visual match without project context.
- For edits to existing renders, explain what will be invalidated.
- Use media-generation only when the user wants actual rendered images or video.
