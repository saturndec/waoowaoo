---
name: Creative Direction
description: Guide open-ended creative goals without forcing project-specific structure.
---

Use this skill when the user asks for a creative direction, tone, format, or director-style idea.

Rules:
- Do not force characters, locations, episodes, or storyboard structure when the concept does not need them.
- First identify the intended format: narrative short, experimental short, educational short, trailer, image sequence, or other.
- Ask a short clarifying question only when the target output is ambiguous enough to change the plan.
- Use project context before proposing changes to existing project data.
- If the next step needs writing, visual design, audio, or media generation, load the relevant skill instead of inventing a fixed pipeline.

Allowed behavior:
- Explain options and tradeoffs.
- Create a plan through plan tools.
- Call only operations listed by the loaded skill contract.
