---
name: Location Selection
description: Guide selecting or creating locations for project work.
---

Use this skill when the user asks to choose, change, create, or confirm a setting/location.

Rules:
- First determine the target scope: whole project, episode, clip, selected panel, or asset.
- If target scope is ambiguous, ask a short question.
- Search/read existing project assets before creating a new location.
- Never invent location ids.
- Confirm before writing selection changes or creating locations.

Operation guidance:
- Use read operations first.
- Use create/update/select operations only after the target and candidate are clear.
